问题

1.人工智能有哪几个主要学派？各自的特点是什么？

符号主义、连接主义和行为主义三大学派。

  • 符号主义学派

    是指基于符号运算的人工智能学派，他们认为知识可以用符号来表示，认知可以通过符号运算来实现。

  • 连接主义学派

    基于神经网络连接机制和学习算法的学派，神经网络模仿自人脑模型。

  • 行为主义学派

    基于控制论和“感知-动作”控制系统的学派。

2.人工智能有哪些主要研究和应用领域？

研究：

机器感知：机器视觉；机器听觉；自然语言理解；机器翻译
机器思维：机器推理
机器学习：符号学习；连接学习
机器行为：智能控制
智能机器：智能机器人；机器智能

应用：

机器博弈；自动定理证明

专家系统；智能决策；智能检索；智能CAD

智能交通；智能电力；智能产品；智能建筑等

3.什么是产生式系统？产生式推理的基本结构由哪几部分组成？

定义：利用产生式式表示方法进行推理的系统

构成：综合数据库、规则库、控制系统

4.什么是状态空间？用状态空间表示问题时，什么是问题的解？

状态空间：一个问题的全部状态和这些状态之间的相互关系构成的集合

问题的解：由初始状态到目标状态所使用的算符序列就是该问题的一个解。

5.什么是语义网络？它有哪些基本的语义关系？

语义网络：用实体及其语义关系来表示知识的有向图

（1）实例关系：ISA

（2）分类关系：AKO

（3）成员关系：A-Member-of

（4）属性关系：Have、Can、Age

（5）包含关系：Part-of

（6）时间关系：Before、After

（7）相近关系：Similar-to、Near-to

（8）位置关系：Loacated-on \at \under\inside\outside

6.什么是框架？框架的一般形式是什么？

框架：通用的可以存储类似以往经验的数据结构

一般形式：

Frame<框架名>

​ 槽名：侧面名 侧面值

7.什么是置换？什么是合一？

置换：形如{t~1~/x~1~，t~2~/x~2~…t~n~/x~n~}的有限集合

合一：使多个谓词变元一致的置换

8.什么是不确定性推理？为什么要采用不确定性推理？不确定推理可分为哪些类型？

不确定性推理：建立在不确定性知识和证据基础上的推理。

为什么：

（1）知识不完备、不精确

（2）知识描述模糊

（3）结论的成因很多

（4）解决方案不唯一

类型：
可信度推理
主观Bayes推理
证据理论
模糊推理
概率推理

9.常见的机器学习方法有哪些？

记忆学习
示例学习
决策树学习
统计学习
集成学习

10.局部择优搜索与全局择优搜索的相同之处和区别是什么？

相同点：都选取了估价函数最小的结点进行扩展

区别：全局择优从Open表所有结点中选择，而局部择优从刚生成的结点中选择

11.什么是自然演绎推理？它所依据的推理规则是什么？

从一组已知为真的事实出发，直接运用经典逻辑中的推理规则推出结论的过程。

主要包括：等价式、永真蕴含式、置换和合一

12.遗传算法的基本思想是什么？基本的遗传操作有哪些？

思想：用模拟生物和人类进化的方法来求解复杂问题。

基本操作：

选择、交叉、变异

模糊知识点

  • 框架

    Unit()填写槽值的书写顺序

    Area()取值范围

    Defalut默认取值

    <>框架

    槽名AKO代表继承

  • 蕴含式的量词一般为全称量词

  • 化子句集的步骤：

    • 转换联结词为析取和合取
    • 将否定符号放进辖域
    • 不同辖域内的相同变元取不同名称
    • 提取量词
    • 消除存在量词
      • 最左边：直接用常量代替
      • 左边有全称量词：用skolem函数代替，左边有几个全称量词，函数就有几个参变量
    • 消除全称量词
    • 消去合取

  • 归纳反演过程中都是析取，需要置换谓词的不同变元

  • 谓词里面除了变元还可以有常量，例如可以用L(x,programing)代表x喜欢编程。

  • 归结反演过程中同一条子句可以多次使用

  • — 若E的出现会增加H为真的可信度，则CF(H, E)>0
    — 若证据的出现减少H为真的可信度，则CF(H, E)<0
    — 若相应证据的出现与H无关，则使CF(H, E) = 0。

  • CF(E) = 1，证据E肯定为真
    CF(E) = -1，证据E肯定为假

    CF(E) = 0，对证据E一无所知

    0 < CF(E) < 1，证据E以CF(E)程度为真
-1 < CF(E) < 0，证据E以CF(E)程度为假

  • 否定证据的可信度：相反数

  • 合取证据取最小，析取证据取最大

  • 合取不确定性取最小，析取不确定性取最大

  • 主观Bayes推理：先求P(H|E),在求P(H|S)

  • A算法八数码难题：g(n)=深度，深度从0开始，h(n)=不在位数码个数

  • A*：g(n)=深度 h(n)=不在位数码移动距离

  • 解树代价：或结点最小，与结点最大或者和代价

  • 端结点非终止结点代价：∞

  • 遗传算法交叉位从1开始数，交叉的对象是两个基因
