决策

  • 决策
    • 定义：决策是为实现一定的目标，在多个备选方案中选择一个方案的分析判断过程
    • 要素：有形要素和无形要素
      • 决策主体
      • 决策制度
      • 决策方案
      • 组织目标
      • 不确定性情景
    • 决策的功能
      • 组织层面：明确组织方向
      • 个人层面：激发组织成员积极性
    • 决策的任务：
      • 外部环境：让组织灵活适应外部环境变化
      • 内部视角：调整和优化组织管理体系
    • 决策的影响因素：
      • 环境
      • 组织历史
      • 决策者的特点
      • 组织文化
    • 决策的准则：
      • 提高效率和效果
        • 重要性原则
        • 准确性原则
        • 灵活性原则
      • 不确定性情景下
        • 乐观
        • 悲观
        • 等概率
        • 最小后悔

环境分析

  • 环境分类
    • 宏观环境：PEST+自然资源
    • 微观环境：
      • 顾客
      • 供应商
      • 竞争者
      • 管制机构
      • 战略伙伴
    • 组织内部环境
      • 物质环境
      • 文化环境
    • 不同层次环境的关系
      • 微观、内部环境比宏观环境更能提供有用的信息
      • 宏观环境对企业的影响通过微观环境体现
      • 微观环境和宏观环境是相对的
      • 微观环境和宏观环境可以转换
  • 分析方法：
    • 宏观PEST
    • 微观波特五力模型
    • 综合分析方法：SWOT分析
    • 环境变化的分析方法：情景分析法（根据情况脚本得出方案脚本）

决策实施

  • 目标管理
    • 含义：鼓励组织成员积极参加工作目标的制定，并在工作中实现自我控制，自觉完成工作任务的管理方法
    • 特点
      • 实行参与管理
      • 重视工作成果而不是工作行为
      • 强调自我控制
      • 建立系统的目标体系
    • 目标管理的过程
      • 目标制定与展开（落实）
      • 目标实施
      • 成果评价

组织结构

  • 形式
    • 直线型组织
      • 特点：从上到下垂直领导
      • 优点
        • 简单
        • 权责关系明确
        • 有序
      • 缺点
        • 专业化水平低
        • 缺乏横向沟通
        • 对管理人员要求高
    • 职能制组织
      • 特点：以专业职能划分部分，管理人员下面设职能机构和人员作为辅助
      • 优点
        • 专业化程度高
        • 减轻管理人员压力
        • 降低管理成本
      • 缺点
        • 缺乏协调
        • 职责不清
        • 不利于通才型管理人员培养
    • 直线职能组织
      • 特点：以直线结构为基础，在各层级设置相应的职能部门
      • 优点：
        • 统一指挥和专业化
        • 减轻管理者负担
      • 缺点
        • 协调难度大
        • 损害下属自主性
        • 降低对环境的适应能力
        • 降低决策效率
        • 增加管理成本

组织文化

  • 影响因素
    • 外部因素
      • 民族文化
      • 制度文化
      • 外来文化
    • 内部因素
      • 领导者素质
      • 组织成员素质
      • 组织发展的阶段
  • 强化认同
    • 广泛宣传
    • 培养典型
    • 加强培训和教育

领导

  • 领导
    • 含义：
      • 名词，领导者，从事领导活动的人
      • 动词：领导，领导行为和过程
    • 三要素：（决定领导的有效性）
      • 领导者
      • 被领导者
      • 情景

激励

  • 激励：组织诱发个体产生某种需要进而促进个体行为与组织目标趋同的管理过程。
  • 激励理论：
    • 马斯洛需要层次理论
      • 人的需要从上到下分5种
        • 生理
        • 安全
        • 社交
        • 尊重
        • 自我实现
      • 低层次需求依靠外部物质，高层次需求依靠内部因素
      • 较低需求满足，高一层次的需求就会产生
      • 低层次需求不会消失，只是对行为的影响变弱
      • 未满足的才有激励作用
    • 双因素理论：
      • 激励因素：让人们感到满意的因素，有关工作本身
      • 保健因素：只能消除不满意，不能让人感到满意，关于工作环境

有效沟通

  • 标准：
    • 保证沟通的量
    • 保证沟通的质
    • 保证沟通信息的时效性
  • 有效沟通的障碍：
    • 人际障碍
    • 组织障碍
    • 文化障碍
  • 克服沟通障碍：
    • 学会倾听
    • 重视反馈
    • 克服认知差异
    • 抑制情绪化反应
  • 冲突
    • 概念：相互作用的主体之间存在的不相容的行为或目标
    • 原因
      • 个人差异
      • 沟通差异
      • 组织结构差异
    • 管理冲突
      • 冲突抑制
        • 竞争
        • 合作
        • 回避
        • 迁就
        • 妥协
      • 冲突激发
        • 冲突合法化
        • 引入外部新鲜血液
        • 组织结构的调整

控制

  • 控制

    • 内涵：对组织内部的管理活动及其效果进行衡量和矫正的过程

    • 原则

      • 有效标准
      • 控制关键点
      • 控制趋势
      • 直接控制
      • 例外原则

    • 方法

      • 层级控制：利用正式的规章制度来规范组织内部的行为并评估绩效
      • 市场控制：借助经济的力量，通过价格机制来规范组织内部的行为
