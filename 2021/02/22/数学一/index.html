反函数

  1. 反函数存在定理：严格单调

复合函数

解题流程：

  1. 广义化：将中间变量u=g(x)带入表达式，例如：f(u)=2u -> f(g(x))=2g(x)
  2. 画图：画自变量的图，注意这里的自变量是中间变量u=g(x)
  3. 求解复合函数（注意分段函数）

函数的四种特性

有界性

定义：对于定义域的一个子集I上，可以找到某一个正数M，使得|f(x)|<=M，称f(x)在I上有界。（解析层面）

注意：

  1. 有界性专门针对某个区间，否则没有意义，不同的区间有界性可能不同。
  2. 几何层面，找到两条直线把函数图像包在其中即为有界。
  3. 还可利用导数来判定有界
单调性

定义：

注意：

  1. 还可利用导数来判定单调性。
  2. 一下函数单调性一致，可用来简化求最值：
    1. $\sqrt{u}$和$\sqrt[3]{u}$，用u代替。
    2. |u|用$u^2$代替。
    3. $\frac1u（u>0）$用u代替。
    4. $\prod\limits_{i=1}^nui$用$\sum\limits_{i=1}^n\ln{ui}$来代替。
奇偶性

定义：

奇函数：f(-x)=-f(x)

偶函数：f(x)=f(-x)

特性：

  1. 奇函数关于原点对称，偶函数关于y轴对称。

  2. 若x=0有定义，奇函数f(0)=0。

  3. f(x)-f(-x)为奇函数，f(x)+f(-x)为偶函数。

  4. 若f’(0)有定义，偶函数f’(0)=0。

  5. 关于x=T对称的充要条件：

    f(x)=f(2T-x)；f(T-x)=f(T+x)

周期性
重要结论 （？）

  1. 可导且具备奇偶性的函数，求导后奇偶性互换。

  2. 可导且具备周期性的函数，求导后周期与原函数相同。

  3. 连续奇函数的所有原函数都是偶函数。

  4. 连续偶函数只有一个原函数是奇函数。

  5. 若连续函数f(x)以T为周期且 ${}\int_{0}^{T}f(x)dx=0{}$则f(x)的所有原函数也以T为周期。

  6. 可导函数在某区间内的导数有界，则原函数在此区间也有界。
