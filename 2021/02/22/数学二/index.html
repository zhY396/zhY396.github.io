  5. 收敛数列的性质

    1. 极限值唯一

      理解：极限值不可能同时趋于两个或以上的值，由定义可得知：假设存在两个值，

      则其中一个值必定在n>N时不能被$\epsilon包围$。

    2. 有界

      理解：由于n是正整数，所以极限一般考虑的是$n\to \infty$的情况，$n无穷大时数列$

      趋于一个值；而n最小为1，不可能产生函数中x=0时候的f(x)趋于无穷大的情况；

      综上收敛数列有界。

    3. 保号性（脱帽）

      理解：极限为0，数列值有可能是在0的附近摇摆的，不一定是恒正，因此没有不等式中没有等于0。

      例如：$\frac{sin\space n}{n}$.

    4. 3.的推论（戴帽）

      理解：数列大于0，极限有可能为0，例如$\frac{1}{n}。$

      而等数列等于0，极限为0。

    5. 极限四则运算规则

      只有极限值分别存在才可能推出运算结果极限存在。

  4. 单调性

    • 判别：导数值

    • 费马定理：(极值存在的必要条件)

      设函数为$f(x)$，已知$f’(x_0)$存在，且$x_0$是极值点，则$f’(x_0)=0.$

      逆命题不成立：导数为0不一定是极值点，例如$f(x)=x^3,f’(0)=0但x=0不是极值点.$

    • 极值存在的第一充分条件

      设函数$f(x)$连续，且在$x_0$的某个去心邻域内可导；

      若在该邻域两侧：$(x_0-\delta,x0),(x_0,x_0+\delta)$内$f’(x)$变号，则$x_0$是极值点，否则$x_0$不是极值点.

    • 极值存在的第二充分条件

      设函数$f(x)$二阶可导，且$f’(x_0)=0,f’’(x_0)\neq 0$，

      则当$f’’(x_0)<0，x_0为极大值点；f''(x_0)>0，x_0为极小值点。$

    • 极值存在的第三充分条件

      设函数$f(x)的n阶导数存在且n为偶数，若f^{(k)}(x_0)(k=0,1,2..n-1)=0,f^{(n)}(x_0)\neq 0，\\则x_0为极值点$

    • 联系导数和原函数的桥梁：拉氏中值定理

    • $f’’(x)=0或f’’(x)$无定义点（分析可能的拐点）
    • 根据这些点划分不同的区间，确定是否为极值点或者拐点，并得出函数的图像。

  2. 渐近线

    • 铅垂渐近线：$\lim\limits_{x\to x_0}f(x)=\infty$，则$x=x_0$叫做铅垂渐近线。注意间断点需要考虑双侧，而渐近线只用考虑单侧，即渐近线不一定位于间断点处。

    • 水平渐近线：$\lim\limits_{x\to \infty}f(x)=y_0$，则$y=y_0$叫做水平渐近线。

    • 斜渐近线：当水平渐近线存在就不需要考虑了；有斜渐近线有可能另一个方向存在水平渐近线。

      当$\lim\limits_{x\to \infty}f(x)=\infty$，则可能出现斜渐近线。

      求斜渐近线(无穷有正有负，可能有两条渐近线）：

      $k=\lim\limits_{x\to \infty}\frac{f(x)}{x}\neq 0$

      $b=\lim\limits_{x\to \infty }f(x)-kx$

      如果k、b有一个不存在，斜渐近线都不存在：可见，$f(x)与x$是同阶无穷大；当k=0变为水平渐近线。

  3. 求区间最值

    1. 求一阶导数不存在或为0的点
    2. 列出区间端点（若为闭区间为端点值，若为开区间为端点的极限值，开区间可以为无穷）函数值和1.所求点的函数值，比较大小即可获得区间最值。

  4. 画函数图像

    1. 确定奇偶性、定义域、无定义点
    2. 求各类点，列表划分定义域区间
    3. 划分单调性和凹凸性，确定极值点，间断点位置处不能确定拐点，连续位置确定拐点。
    4. 求渐近线

    $f(x)在闭区间[a,b]连续，开区间(a,b)可导，且f(a)=f(b)\Rightarrow \exists \xi \in (a,b),f’(\xi)=0.$

    注：柯西中值定理不能通过单独对$f(x)、g(x)$运用拉氏定理得到，因为$\xi$是一个数而不是两个.

    证明方法：将式子整理为:$(f(b)-f(a))\cdot g’(\xi) -(g(b)-g(a))\cdot f’(\xi)=0$，然后构造辅助函数$F(x)$使得$F(a)=F(b)，同时F’(x)等于上式左边的部分，再用罗尔定理即可。$
