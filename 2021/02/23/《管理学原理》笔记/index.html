管理导论

管理的概念

案例

华为的管理：

  1. 轮值CEO、轮值董事长
  2. 高薪、股权激励
  3. 客户至上，研发创新高投入

日本资源缺乏，依靠管理取得了世界经济体量第二的卓越地位，而巴西资源丰富，却从发达国家降到发展中国家。

对管理的理解
  1. 亨利.法约尔：管理的五项职能：计划、组织、指挥、协调、控制
  2. 西蒙：管理就是决策
管理的概念

管理就是管理者为了实现组织的共同目标，在特定的时空中，对组织成员在目标活动中的行为进行协调的过程。

概念中强调过程，过程包含两方面：职能和主要活动。

职能包括：

计划、组织、领导、控制、效率、效果

管理概念的解析
  1. 管理是为了实现组织目标，组织目标由效率和效果来衡量。
  2. 管理具有情境依赖性，不可照搬管理手段和措施。
  3. 管理核心是人的行为。
  4. 管理的本质是协调。
管理的实质

管理不是某种方法或者手段，而是管理者的意识和思维方式。包括：整体意识、贡献意识、成功意识等。

管理的职能和性质

管理的过程
  1. 决策与计划：确定目标和战略
  2. 组织：组织结构的设计和人员的配备
  3. 领导：指导和激励所有参与者，并解决成员的冲突
  4. 控制：对活动进行监控，以确保按计划执行
  5. 创新：打破固有思维，发现新问题

决策（计划）、组织、领导、控制各职能相互交叉渗透

管理的特征
  1. 科学性：管理具有内在规律（提炼各种管理原则）
  2. 艺术性：管理的情境依赖性（各国调整房价）

管理者的角色和技能

明茨伯格的十角色理论
  1. 人际关系方面：
    1. 挂名首脑：象征
    2. 领导者：激励和动员下属
    3. 联络者：维护外部和内部联系网络
  2. 信息传递方面：
    1. 监听者：获取信息
    2. 传播者：传播信息
    3. 发言人：向外部发布信息
  3. 决策制定方面
    1. 企业家：快速识别和把握机会，发起变革
    2. 混乱驾驭者：组织动乱补救措施
    3. 资源分配者：分配各项资源
    4. 谈判者：组织代表进行谈判

注：不同企业的角色重要性不同

管理者的技能
  1. 概念技能：透过现象看本质，抽象问题
  2. 人际技能：打交道、沟通能力
  3. 技术技能：专业领域的技能

注：不同层次管理人员对技能的要求不同，高层管理者要求高概念技能，中层管理者兼有概念技能和人际技能，而基层管理者侧重人际技能和技术技能。

成功的管理者和有效的管理者

管理者从事四项活动：

  1. 传统管理
  2. 沟通：工作内容信息交流
  3. 人力资源管理
  4. 网络关系：社交、政治活动

成功的管理者在网络关系中比重大，有效的管理者在沟通中比较大，可见社交和政治活动对提升有较大影响。

决策

华为重要决策案例分析

  1. 有哪些决策

    1. 上市融资还是开放股权？
    2. 专攻营销还是专攻研发？
    3. 做平台还是做产品？
    4. 轮值领导还是专权领导？

  2. 请分析这些决策的类型，决策所指向的问题以及决策目标，并对决策优劣进行评价（至少选择两个分析）。

    分析1.上市融资还是开放股权？

    决策类型：长期决策、战略决策、集体决策

    决策目标：激励员工、促进体制改革、保持企业纯净

    优劣评价：华为选择了开放股权的决策，不上市融资，这样的劣处是降低了企业的总资本，优点是刺激了员工的积极性，保证了企业的纯净，没有鱼龙混杂的资本注入。

    分析2.专攻营销还是专攻研发？

    决策类型：长期决策、战略决策、集体决策

    决策目标：选择企业发展方向、确定愿景

    优劣评价：华为选择了专攻研发的决策，投入大量资本到研发当中，这样的劣处是降低了短期企业的总收入，优点是细水长流，韬光养晦，研发出长期竞争力较强的技术。
